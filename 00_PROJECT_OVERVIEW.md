## 1. 프로젝트 한 줄 정의

음식 사진을 업로드하면 AI가 메뉴·칼로리·탄단지를 자동 분석하고, 그룹원들과 하루 식단을 실시간으로 비교·공유할 수 있는 **그룹형 AI 식단 관리 웹 앱**.

---

## 2. 핵심 가치 (절대 훼손하지 말 것)

| 가치 | 설명 |
|------|------|
| **사진 한 장으로 끝** | 직접 음식명을 검색하거나 칼로리를 입력하지 않아도 됨 |
| **AI 자동 분석** | CNN 모델이 음식 종류·칼로리·탄단지를 예측 |
| **그룹 비교** | 친구/가족과 하루 섭취량을 실시간으로 경쟁·공유 |
| **개인 맞춤** | 키·몸무게·활동량·목표 기반 맞춤 권장량 제공 |

## 6. 핵심 비즈니스 로직 요약

### 하루 기준 계산
```
하루 시작: 당일 04:00:00
하루 종료: 익일 03:59:59
→ 예) 2026-05-24 04:00 ~ 2026-05-25 03:59 = "5월 24일 식단"
```

### 칼로리 목표 계산 (Mifflin-St Jeor 공식)
```
남성 BMR = 10 × 체중(kg) + 6.25 × 키(cm) - 5 × 나이 + 5
여성 BMR = 10 × 체중(kg) + 6.25 × 키(cm) - 5 × 나이 - 161

TDEE = BMR × 활동계수
  - 비활동적(1.2), 가벼운 활동(1.375), 보통(1.55), 활발(1.725), 매우 활발(1.9)

목표 칼로리:
  - 감량: TDEE - 500 kcal
  - 유지: TDEE
  - 증량: TDEE + 300 kcal
```

### AI 분석 흐름
```
사진 입력
  → 이미지 전처리 (224×224 리사이즈, 정규화)
  → MobileNetV2 추론 (Top-3 예측 반환)
  → 음식 DB 매칭 (food_items 테이블)
  → 칼로리/탄단지 계산
  → 사용자 확인 및 수정 가능
  → DB 저장 (is_edited 플래그 포함)
```
