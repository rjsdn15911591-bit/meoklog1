```sql

```
서버 시작(lifespan) 시 자동 실행, 이미 컬럼이 있으면 무시.

### 2-2. AI 칼로리 보정 로직

## 3. 버그 수정

### 3-1. 댓글 0개로 표시되는 버그 (500 에러)

`backend/app/routers/meals.py` — `get_comments` 수정.

**원인:** async SQLAlchemy 환경에서 `comment.user` 관계를 lazy loading으로 접근 →
`MissingGreenlet` 예외 → 500 에러. 그룹 피드에서는 댓글 수(count)가 별도 쿼리라 17개 표시,
상세 페이지에서는 실제 댓글 객체를 직렬화할 때 실패.

**수정:**
```python
result = await db.execute(
    select(Comment)
    .options(selectinload(Comment.user))  # eager loading 추가
    .where(Comment.meal_id == meal_id)
    .order_by(Comment.created_at)
)
```

### 3-2. 리액션 카운트 즉시 반영 안 되는 문제

`app/meal/[mealId]/page.tsx` — `reactionMutation` 수정.

**원인:** API 응답이 완료된 뒤 쿼리 무효화(invalidateQuery)로 카운트 갱신 →
이모지 플로팅 애니메이션(1.8s) 이후에야 숫자 변경.

**수정:** `onMutate` 낙관적 업데이트 추가 — 클릭 즉시 로컬 캐시에서 카운트 +1,
실패 시 `onError`에서 이전 값으로 롤백.

### 3-4. 달력 높이 점프 문제 (5주 ↔ 6주)

`components/ui/DatePickerModal.tsx` 수정.

**원인:** 월에 따라 달력 행이 5개(35일)이거나 6개(42일)이어서 높이가 달라짐.

**수정:** 항상 42일(6×7) 고정 그리드 렌더링.
```typescript
const gridStart = startOfWeek(startOfMonth(viewMonth), { weekStartsOn: 0 });
const days = eachDayOfInterval({ start: gridStart, end: new Date(gridStart.getTime() + 41 * 86400000) });
```

### 3-5. 그룹 비교/랭킹 페이지 충돌 (`toLocaleString` on undefined)

`backend/app/routers/groups.py` 수정.

**원인:** 백엔드 응답 키 `today_calories` → camelCase 변환 → `todayCalories`.
프론트엔드 타입은 `totalCalories`. 키 불일치로 undefined.toLocaleString() 크래시.

**수정:** 백엔드 응답 키를 `total_calories`로 통일 → camelCase 변환 후 `totalCalories`.
`entries.sort` 및 평균 계산도 동일 키 사용으로 수정.

---

### 3-6. 한국 표준시(KST) 시간 표시 오류

`frontend/lib/utils.ts` — `formatTime()` 수정.

**원인:** 백엔드가 UTC naive datetime(Z 없음)을 반환 → 브라우저가 로컬 시간으로 파싱
→ 서버 UTC 기준 그대로 표시 (한국 시간 -9시간).

**수정:**
```typescript
export function formatTime(dateString: string): string {
  const normalized =
    dateString.endsWith('Z') || dateString.includes('+') ? dateString : dateString + 'Z';
  const date = new Date(normalized);
  return new Intl.DateTimeFormat('ko-KR', {
    hour: '2-digit', minute: '2-digit', hour12: false, timeZone: 'Asia/Seoul',
  }).format(date);
}
```

`frontend/lib/utils.ts` — `formatCalories()` 수정.

if (calories == null || isNaN(calories)) return '0';
